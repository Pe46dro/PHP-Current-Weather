# PHP Current Weather

A simple PHP class to get current whaeter of place using:

  - [CSS Weather Icons](http://erikflowers.github.io/weather-icons/) show weather
  - [Open Weather Map](http://openweathermap.org/)  weather information

### Version
1.0

### Use

```php
include('src/class.weather.php');
$var = new Weather("Venice","IT");
```

then you can use all method to access data for the complete list check in API folder
